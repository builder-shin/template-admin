# types

## Purpose
프로젝트 전반에서 사용되는 TypeScript 타입 정의 디렉토리.

## Key Files

| File | Description |
|------|-------------|
| `index.ts` | 공통 타입 정의 및 re-export (NavItem 등) |
| `base-form.ts` | 폼 컴포넌트 기본 타입 정의 |
| `data-table.ts` | 데이터 테이블 관련 타입 정의 |

## For AI Agents

### Working In This Directory
- 새 타입은 관련 파일에 추가하거나, 독립적인 도메인이면 새 파일을 생성한다
- 2개 이상의 모듈에서 공유되는 타입만 이 디렉토리에 배치한다
- 단일 모듈에서만 사용하는 타입은 해당 모듈 내에 정의한다
